# Online Image & Video Recognition Chrome 扩展安装教程

## 1. 准备文件

1. 将扩展文件夹完整放到本地电脑，例如：`C:\Users\Administrator\Documents\NEW`。
2. 确认文件夹里有 `manifest.json`、`background.js`、`content.js`、`popup.html`、`options.html` 等文件。
3. 不要只选择某个子文件夹，也不要选择压缩包；Chrome 需要加载“已解压”的完整扩展目录。

## 2. 安装到 Chrome

1. 打开 Chrome 浏览器。
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择扩展所在文件夹：`C:\Users\Administrator\Documents\NEW`
6. 页面出现 `Online Image & Video Recognition` 卡片，即表示安装成功。
7. 建议点击浏览器右上角扩展图标，把 `Online Image & Video Recognition` 固定到工具栏。

## 3. 配置 API

1. 在 `chrome://extensions/` 找到 `Online Image & Video Recognition`。
2. 点击“详情”或扩展图标里的“设置”。
3. 填写识别模型服务商、`Base URL`、模型名和 `API Key`。
4. 如果需要生图预览，也要在设置页配置生图预览服务商和对应模型。
5. 保存设置后，刷新正在使用的网页。

常用配置示例：

- Gemini：`https://generativelanguage.googleapis.com/v1beta`
- OpenAI：`https://api.openai.com/v1`
- OpenAI 模型示例：`gpt-4.1-mini`
- Gemini 模型示例：`gemini-2.5-flash`

## 4. 基础使用

1. 打开一个包含图片或视频的网页。
2. 鼠标移动到图片或视频上方，左上角会出现工具按钮。
3. 点击 `识图`：反推图片提示词。
4. 点击 `识视频`：按 5s、10s、15s、25s、30s、35s 等时长抽帧分析。
5. 识视频结果里可以查看即梦提示词、分镜结构、逐帧口播文案、抽帧缩略图。
6. 重点能力：可以修改故事大纲，让后续分镜按新的内容方向同步调整。
7. 点击“打开分镜编辑器”，可以继续编辑主体、场景、风格、BGM、镜头时间、画面描述、镜头语言和资产图。

## 5. 更新扩展

1. 替换本地扩展文件夹里的新文件。
2. 打开 `chrome://extensions/`。
3. 找到 `Online Image & Video Recognition`。
4. 点击卡片右下角的刷新按钮。
5. 刷新目标网页后再使用。

## 6. 常见问题

### Service Worker 显示“无效”

1. 先点击扩展卡片上的刷新按钮。
2. 如果仍无效，点击“错误”查看具体报错。
3. 常见原因是文件没放完整、代码语法报错、或加载了错误目录。

### 页面上没有出现按钮

1. 刷新当前网页。
2. 确认扩展已开启。
3. 确认鼠标悬停在真实的图片或视频区域。
4. 部分 iframe、加密视频或特殊站点可能会限制脚本注入。

### API 报额度不足

1. 这通常是模型服务商额度或限流问题。
2. 可以等待额度恢复，或在设置页更换 API Key、模型、服务商。
3. ChatGPT Plus 额度不等于 OpenAI API 额度，API 需要单独开通计费。

### 视频下载失败

1. 浏览器扩展只能直接下载可访问的媒体文件。
2. `blob:`、`m3u8`、加密分片、DRM 视频不一定能直接下载为 MP4。
3. 如果只是录制当前播放画面，可能会受跨域 Canvas 限制。

## 7. 卸载扩展

1. 打开 `chrome://extensions/`。
2. 找到 `Online Image & Video Recognition`。
3. 点击“移除”。
4. 如果要彻底清理，也可以删除本地扩展文件夹。
