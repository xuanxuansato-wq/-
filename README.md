# Online Image & Video Recognition

Online Image & Video Recognition 是一个 Chrome Manifest V3 扩展，用于网页图片/视频提示词反推、视频下载辅助，以及本地视频标识与剪辑项目整理。

## 主要功能

- 鼠标悬停网页图片，点击 `识图` 反推图片提示词。
- 鼠标悬停网页视频，点击 `识视频`，可从当前播放位置截取 5s、10s、15s、25s、30s 或 35s 片段进行分析。
- 输出中文即梦提示词、关键词、镜头线索等内容。
- 视频结果面板保留重新分析、复制提示词并打开即梦、下载视频等快捷操作。
- 点击扩展图标可打开“视频标识码与快速剪辑”工具。

## 视频标识与剪辑工具

打开扩展图标后进入 `merge.html` 工具页：

- 给成片添加图片标识，位置保留顶部居中和底部居中。
- 标识图片独立一列上传，可选择顶部居中或底部居中。
- 可把多个视频直接拖拽到页面上传区，按片段顺序组合成一条片。
- 视频上传后会直接在上传页生成可播放预览，便于快速确认素材。
- 支持合并成片，也支持逐段批量导出。
- 可开启视频高清增强，参考 `/upscale` 的“保构图、保主体、真实高分辨率交付”思路，按输出分辨率或长边 1920/2560/3840px 进行本地高质量放大，并做保守清晰化；配置页支持原始帧/增强帧对比预览。
- 可添加文字标识语，直接编辑标识语内容、透明度、大小和 X/Y 位置，也可在预览区域拖动定位。
- 每段视频上传后在当前窗口预览，剪辑区间叠在视频预览底部，可直接拖动或缩放，也可去掉。
- 可一键识别尾部落版/Logo 动画，自动把片段结束点裁到疑似落版开始处；找不到明显落版时，可按设置秒数裁掉视频末尾。
- 去落版裁切条使用类似剪辑软件的片段条，包含文件名、缩略图、波形、裁切边界和播放头。
- 输出时会在浏览器本地实时渲染视频并叠加标识图片，合并模式输出一个 WebM 文件，批量模式按片段输出多个 WebM 文件。
- 支持撤销、重做、导出项目 JSON、导入项目 JSON。
- 所有剪辑都是非破坏式参数，源视频不会被修改。

## 安装

1. 打开 `chrome://extensions/`。
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择本文件夹。
5. 打开扩展设置页，填写识别服务、Base URL、模型名和 API Key。

## 默认模型

Gemini:

- Base URL: `https://generativelanguage.googleapis.com/v1beta`
- Model: `gemini-2.5-flash`

OpenAI Compatible:

- Base URL: `https://api.openai.com/v1`
- Model: `gpt-4.1-mini`

不同网关可能使用不同模型名，也可能不支持 `/chat/completions` 的图片输入。需要时请在设置页调整。

## 隐私

API Key 只保存在本机 Chrome storage。图片和抽帧视频会直接发送给你配置的模型服务商。本扩展不使用项目自有服务器。

## 限制

视频下载只支持可直接访问的媒体文件。`blob:`、`MediaSource`、`m3u8`、DRM、加密分片流等内容可能无法直接保存为完整 MP4。

本地视频批处理使用浏览器 Canvas 与 MediaRecorder 渲染，Chrome/Edge 通常输出 WebM；如果必须交付 MP4，可用后续转码工具把 WebM 转为 MP4。

视频高清增强为浏览器本地逐帧处理：高质量缩放、微对比/饱和度增强和轻量锐化。它遵循保守修复逻辑，不会重绘或生成新内容；如果需要 Real-ESRGAN/Topaz 级别的 AI 超分，可继续接入本地或云端超分后端。
