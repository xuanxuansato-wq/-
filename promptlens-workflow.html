🧭 使用步骤

Step 1

配置 API

在设置页填写识别 API 与生图预览 API；两套配置互不影响，额度不足时可切换服务商。

输出：可用模型环境
Step 2

打开素材页

进入广告素材、图片或视频页面，悬停目标内容后会出现识别、下载、设置等快捷按钮。

输出：待识别素材
Step 3

识图 / 识视频

图片直接反推中文提示词；视频可选择 5s、10s、15s、25s、30s、35s 节选并抽取关键帧。

输出：提示词 / 分镜初稿
Step 4

编辑结构

视频先统一修改主体、场景、风格，再逐镜头调整镜头时间、画面描述、BGM、镜头语言。

输出：可复用分镜表
Step 5

预览与复用

选择 1:1、9:16、16:9、3:4、4:3 等比例生成预览图，或复制即梦中文提示词继续创作。

输出：生图预览 / 即梦 Prompt
Step 6

下载归档

下载识别图片、预览图、视频素材与分镜内容，沉淀为本地素材库，方便后续批量复盘。

输出：本地素材资产

图片反推与生图预览

图片可打开大图、下载原图、编辑中文提示词，并选择常用比例生成预览。

识别模型和生图预览模型分开配置，额度不足时可以快速切换 GPT、Gemini、豆包或即梦。

📌 产出结构

图片反推

  • ①上传或选中图片，生成中文生图提示词。
  • ②在弹窗内直接修改提示词，复制或调用 API 生成预览。
  • ③缩略图可点开查看正常尺寸，并支持直接下载。

视频分镜

  • ①按选定秒数抽关键帧，秒数越长抽帧越多。
  • ②先编辑主体、场景、风格，再修改每个镜头细节。
  • ③镜头时间从 0 开始，点击时间可跳到对应视频帧。
