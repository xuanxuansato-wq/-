1

安装扩展：加载本地文件夹

不要选择压缩包，也不要选择子目录；需要选择完整扩展根目录。

  1. 打开 Chrome / Edge 地址：chrome://extensions/
  2. 右上角开启“开发者模式”。
  3. 点击“加载已解压的扩展程序”。
  4. 选择扩展根目录：C:\Users\Administrator\Documents\NEW
  5. 看到 Online Image & Video Recognition 卡片后，建议固定到浏览器工具栏。

配置 API：识别与生图分开设置

API Key 只保存在本机 Chrome storage；ChatGPT Plus 不等于 OpenAI API 额度。

  1. 在扩展卡片点击“详情 / 选项”，或在网页工具条点“设置”。
  2. 配置“识别模型”：用于识图、识视频、分镜反推、按图反推镜头字段。
  3. 配置“生图预览”：用于图片提示词预览图，可与识别模型使用不同 Key。
  4. 保存后，回到素材网页刷新页面再使用。
OpenAI / GPT Gemini 豆包 即梦 OpenAI Compatible
  • Gemini Base URL：https://generativelanguage.googleapis.com/v1beta
  • OpenAI Base URL：https://api.openai.com/v1
3

网页操作：悬停后识图 / 识视频 / 下载

安装或更新后必须刷新目标网页，旧页面里的内容脚本不会自动更新。

  1. 打开素材网页，鼠标悬停到目标图片或视频上。
  2. 点击 识图：反推中文提示词，可编辑后复制，也可生图预览。
  3. 点击 识视频：从当前播放点抽关键帧，反推分镜脚本。
  4. 点击 下载：优先直接下载可访问 MP4；受限视频会提示原因。
  5. 识视频建议先播放 5–10 秒，让网页加载到目标清晰度。
5s 10s 15s 25s 30s 35s 更多：45s / 1min
4

识图结果：编辑提示词并生成预览

图片缩略图可点开查看或下载；提示词可以直接改成需要的主体、场景和风格。

  1. 点击图片工具条里的 识图。
  2. 查看中文提示词，直接在结果框里修改。
  3. 选择常用比例：9:16、16:9、3:4、4:3。
  4. 点击“生图预览”，使用设置页配置的生图 API。
  5. 复制最终提示词，可用于即梦等生成工具。
5

分镜编辑器：整理脚本和资产图

适合内部对接：全局参考、分镜图、资产图、镜头字段都能继续编辑。

  1. 在识视频结果中点击“打开分镜编辑器”。
  2. 全局参考：主体、场景、风格都可拖拽/点击上传图片；主体支持多图。
  3. 主体图上传后，会自动同步到匹配镜头的资产图。
  4. 每个镜头可编辑：镜头时间、画面描述、镜头语言。
  5. 分镜图 / 资产图支持拖拽上传、选中、删除；资产区有“删除选中资产”。
  6. 点击镜头时间旁的跳转，可回到原视频对应帧；“打开原页面”会尝试定位原视频。
6

导出交付：PNG 图片与可编辑 HTML

PNG 用于快速发图；HTML 用于后续继续修改文字、替换图片和二次交付。

  1. 点击 生成脚本图片：直接下载完整 PNG，参考图和镜头图会完整展示。
  2. 点击 下载可编辑文件：下载本地 HTML。

  5. 修改完成后，点击 HTML 顶部“下载当前修改版 HTML”。

更新扩展：每次改完代码后这样操作

这一步很关键，否则旧页面可能还在运行旧脚本。

  1. 打开 chrome://extensions/。
  2. 找到 Online Image & Video Recognition，点击卡片右下或右上角“刷新”。
  3. 回到目标素材网页，按 Ctrl + R 刷新页面。
  4. 重新悬停图片或视频再操作。

一句话流程

加载扩展 → 设置 API → 刷新素材网页 → 悬停识图/识视频 → 打开分镜编辑器 → 上传参考图和资产图 → 生成脚本 PNG 或下载可编辑 HTML。
