在线识图识视频｜Chrome 插件安装说明

用于图片反推提示词、视频分镜反推、分镜编辑器、提示词复制和视频辅助下载。重点支持修改故事大纲，并按新的内容方向同步调整后续分镜描述，适合视频创作者拆解素材并改写成自己的创意脚本。

版本 0.2.87

准备文件

当前可上传/分发的插件压缩包：

C:\Users\Administrator\Documents\NEW\Online-Image-Video-Recognition-0.2.87.zip

本地调试也可以直接使用源码目录，目录内需要能看到 manifest.json。

C:\Users\Administrator\Documents\NEW

方式一：本地安装测试

1

打开扩展管理

Chrome 地址栏输入 chrome://extensions/。

2

开启开发者模式

右上角打开 Developer mode / 开发者模式。

3

加载插件文件夹

点击 Load unpacked / 加载已解压的扩展程序，选择源码目录或解压后的文件夹。

4

固定插件图标

点击浏览器右上角拼图图标，把“在线识图识视频”固定到工具栏。

方式二：上传 Chrome Web Store

打开开发者后台
Add new item
上传 zip 包
填写商店信息
Submit for Review

开发者后台地址：https://chrome.google.com/webstore/devconsole。需要你的 Google 开发者账号登录，首次发布可能需要完成开发者注册。

基础使用流程

  • 打开素材页面后，左上角会出现插件按钮：识视频 / 下载 / 设置。
  • 点击 识视频，选择 1s、5s、10s、15s、30s、45s 或 1min 进行抽帧识别。
  • 重点：识别完成后可修改故事大纲，让后续分镜按新的内容方向同步调整。
  • 也可以进入 打开分镜编辑器 继续修改分镜脚本、图片和资产图。
  • 分镜编辑器支持上传/拖拽分镜图和资产图，并可下载可编辑 HTML 或脚本图片。
  • 点击 设置 可配置 Gemini、OpenAI/GPT、豆包、千问、即梦或兼容 API。

常见问题

Service Worker 无效

通常是代码报错或旧版本缓存。先点扩展页的刷新按钮，再重新打开网页测试。

Unexpected token <

接口本来应返回 JSON，但实际返回了 HTML 页面。检查 API 地址、服务是否启动、是否被登录页/404 拦截。

API 地区或额度报错

说明当前服务商不可用、额度不足或地区不支持。到设置页切换 API Key、模型或服务商。
